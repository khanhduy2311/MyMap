# Quick Start Commands

## Development
```bash
npm install
cd MindMapBoDoi/project-d10
npm install
npm run build
cd ../..
npm start
```

## Deploy to Render
See [DEPLOY_RENDER.md](./DEPLOY_RENDER.md) for detailed instructions.

## Environment Variables Required
- `MONGO_URI`
- `SESSION_SECRET`
- `GEMINI_API_KEYS`
- `OCRSPACE_API_KEY`
- `HUGGINGFACE_TOKEN`
- `OPENROUTER_API_KEY`
- `REDIS_URL` (auto-configured on Render)
